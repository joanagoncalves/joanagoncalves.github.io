Representation Learning from Directed Graphs

Representation learning is the machine learning task of automatically discovering features or representations of the data, which can be used for modelling, clustering and classification. The goal of this project is to: (i) investigate representation learning approaches for encoding the structure of directed networks/graphs so that it can be easily exploited by machine learning models, and (ii) assess their performance against commonly used measures based on a limited predefined number of network features. With application to regulatory data, including transcription factor-target binding and eQTL data.

network representation learning, machine learning, deep learning, graphs, graph theory
